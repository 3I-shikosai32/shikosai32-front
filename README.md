# shikosai32-front

## Setup

1. リポジトリをクローンする。
   ```powershell
   git clone "https://github.com/3I-shikosai32/shikosai32-front"
   cd "shikosai32-front"
   ```
2. 依存パッケージをインストールする。
   ```powershell
   yarn install
   ```
3. 責任者から環境変数を受け取り、それをもとに`.env`ファイルを作成する。
4. サーバーを起動し、手順が上手くいっていることを確認する。
   ```powershell
   yarn dev
   ```
